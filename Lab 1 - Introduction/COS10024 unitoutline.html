Aims

This unit is designed tointroduce the technology of the Internet and World Wide Web and to develop an understanding of the technologies assoiated with programmig for the World Wide Web

Learning Objectives

After successfully completing this unit , you should be able to:

      1. Develop web pages using HTML , Javascript and CSS.
      2.Demonstrate an understanding of key technologies of the World Wide Web.
      3.Describe the protocols of the World Wide Web.
      4. Design effective and user-friendly pages using usability principles and style guides.
      5. Demonstrate an understanding of history of the Internet and the World Wide Web.

Content

  • Web Concepts: definition , history and fundamental concepts
  • Introduction to HTML, CSS , Web Developement processes
  • HTML5: logical document structure , syntax , images , links , maps , tables , frames , forms.
  • W3C standards and validation
  • Design , Layout and Usability: principles of naviation , usability , page style guides , standards.
  • Protocols and sever technology: HTTP , TCP/IP , MIME , URI , DNS
  • JavaScript: syntax , object model , event handling , common tasks
  • Interactivity: DOM , and some evolving technologies forms processing , common tasks
  • Style sheets: fundamentals , CSS formatting , CSS positioning , standards
  • Introduction to XML
        1. Syntax
        2. DTDs
        3. XSL
        4. XML Applications
  • Web design and usability: principles of navigation , usability , style guides , standards
